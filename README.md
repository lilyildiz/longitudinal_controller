
# ROS2 Longitudinal Controller

A basic PID Longitudinal Controller for ROS2 Foxy and SVL Simulator.


## Dependencies

`lgsvl-msgs`

## Deployment

To deploy this project run 
```bash
  ros2 run longitudinal_controller controller
```
